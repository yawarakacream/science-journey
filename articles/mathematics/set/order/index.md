~~~definition:順序集合

集合 $A$ に関係 $\leq$ が定義されているとき，

$A$ が **順序集合**

$$
\overset{def}\iff
\begin{cases}
    a \leq a \quad \left( {}^{\forall} a \in A \right) \\
    a \leq b,\ b \leq a \implies a = b \quad \left( {}^{\forall} a,\ b \in A \right) \\
    a \leq b,\ b \leq c \implies a \leq c \quad \left( {}^{\forall} a,\ b,\ c \in A \right)
\end{cases}
$$

$A$ が **全順序集合**

$\quad \overset{def}\iff$ $A$ が順序集合でかつ，$a \leq b$ または $b \leq a ~ ({}^{\forall} a,\ b \in A)$

~~~
